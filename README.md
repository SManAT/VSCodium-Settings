## Intro

A tool to Backup and Restore Settings and Extensions for [VSCodium](https://vscodium.com/).

## Usage

### Windows

```powershell
powershell.exe -NoProfile -ExecutionPolicy Bypass -File "%~dp0VSCodiumSync.ps1"
```

or just 

```powershell
./VSCodiumSync.ps1
```

### Unix

```bash
./VSCodiumSync.sh
```
